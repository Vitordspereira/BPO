2. No Front-end (HTML/JavaScript)
Para que o usuário veja os pontos aparecendo enquanto digita, você pode usar uma máscara no campo do formulário.

Se você estiver usando JavaScript puro no seu HTML, adicione este evento ao seu campo de input:

HTML
<input type="text" id="numero" maxlength="19" oninput="mascararCartao(this)" placeholder="0000.0000.0000.0000">

<script>
function mascararCartao(input) {
    let v = input.value.replace(/\D/g, ""); // Remove tudo que não é dígito
    v = v.replace(/(\d{4})(?=\d)/g, "$1."); // Adiciona o ponto a cada 4 dígitos
    input.value = v;
}
</script>
